## 1. デザインコンセプト

**「体験への案内係」を、藤色の糸でつなぐ。**

- Airbnb から借りるもの: 写真主導のカード、角丸の温かい UI、体験を探す→詳細→申込方法という迷わない導線、余白による信頼感
- このサイト固有のもの: 日本の伝統色（藤色 × 生成り × 藍）、明朝体の見出し、そして「糸」のモチーフ
- 対象読者は県外観光客・訪日客・地元の方・高齢の担い手。**誰が見ても威圧感がなく、温かく、迷わない**ことを最優先する

### シグネチャー要素: 「糸と結び目」
サイトの使命「点と点を線で結ぶ」を視覚言語にする。

- セクション区切りは水平線ではなく、**1px の細い糸（藤色）に小さな結び目（4px の丸）を打った線**
- 工程ページのステップは、縦に垂れる 1 本の糸が各工程（結び目）を貫くタイムライン表現
- カレンダーのイベントドットも同じ「結び目」— 日々の点が糸で結ばれてゆくメタファー
- 装飾はこの 1 系統に絞る。グラデーション・パターン背景・過剰な花飾りは使わない（Sayo's Journal の水彩花柄とは差別化し、こちらは「工芸の静けさ」）

| `--color-primary-400` | `#9A8CC4` | 藤色 | バッジ、アイコン、装飾 |

| トークン | Hex | 用途 |
|---------|-----|------|
| `--color-success` | `#3D7A5C` | 受付中バッジ |

### コントラスト規約
- 本文 `#2B2926` on `#FAF7F0` = 12.9:1 ✓

- `#9A8CC4`（藤色）は**大きな装飾・アイコンのみ**に使用し、本文テキストには使わない

## 3. タイポグラフィ

### フォントファミリー
| ロール | 日本語 | 英語 | 備考 |
|--------|--------|------|------|
| ディスプレイ（h1/h2） | **Shippori Mincho** (600) | **Cormorant Garamond** (600) | Sayo's Journal との姉妹感。EN は locale で自動切替 |
| 見出し（h3/h4） | Shippori Mincho (500) | Cormorant Garamond (600) | |
| 本文 | **Noto Sans JP** (400) | **Inter** (400) | 読みやすさ最優先 |
| UI・メタ情報 | Noto Sans JP (500) | Inter (500) | ボタン、バッジ、日付 |

- next/font で全て self-host（サブセット化。Shippori Mincho は使用ウェイトのみ）
- `[locale]` に応じて `font-family` スタックを CSS 変数で切替（`--font-display` / `--font-body`）

### タイプスケール
| トークン | サイズ / 行間 | 用途 |
|---------|--------------|------|
| `display` | 40px / 1.35（SP: 30px） | ページ Hero 見出し |
| `h2` | 28px / 1.45（SP: 24px） | セクション見出し |
| `h3` | 20px / 1.5 | カードタイトル、小見出し |
| `body-lg` | 18px / 1.9 | 歴史・物語などの読み物本文 |
| `body` | 16px / 1.8 | 標準本文 |
| `caption` | 13px / 1.6 | メタ情報、注記 |

- 日本語本文の行間は 1.8〜1.9 を確保（Airbnb の 1.5 前後は和文には詰まりすぎるため上書き）
- 英語ページは行間 1.6 に自動調整
- 読み物セクション（歴史・物語）の最大幅は 65ch 相当（720px）

## 4. 形状・余白・エレベーション

Airbnb の「角丸で温かい」文法を踏襲。

| トークン | 値 | 用途 |
|---------|-----|------|
| `--radius-sm` | 8px | 入力欄、タグ |
| `--radius-md` | 12px | カード内画像 |
| `--radius-lg` | 16px | カード |
| `--radius-full` | 9999px | ボタン（ピル型）、バッジ |
| `--shadow-card` | `0 1px 2px rgba(43,41,38,.06), 0 4px 12px rgba(43,41,38,.08)` | カード標準 |
| `--shadow-hover` | `0 2px 4px rgba(43,41,38,.08), 0 8px 24px rgba(43,41,38,.14)` | カード hover |

- スペーシングは 4px グリッド。セクション間は 80px（SP: 56px）
- コンテンツ最大幅: 1120px。カードグリッドは 3 列（Tablet 2 列 / SP 1 列）

## 5. コンポーネント設計

### 5.1 ボタン
- **Primary**: 藤紫 `#6D5BA4` 背景 + 白文字、ピル型、高さ 48px、hover で深藤 `#5B4A8A`
- **Secondary**: 白背景 + 墨文字 + border、ピル型
- **申込ボタン**（イベント詳細）: Primary を使用。ラベルは「申し込む（外部サイト）」等、**遷移先が外部であることを明示**

### 5.2 工芸カード（/crafts, ホーム）
Airbnb のリスティングカード踏襲。

```
┌──────────────────┐
│  写真 (4:3, radius-md)      │
│                              │
├──────────────────┤
│ 遠山ふじ糸          [藤色タグ]│ ← h3 (Shippori Mincho)
│ 📍 飯田市南信濃（遠山郷）      │ ← caption
│ 山藤の蔓から糸を紡ぐ          │ ← body, 1行
└──────────────────┘
```
- hover: shadow-hover + 写真をゆるやかに scale(1.03)（transition 300ms）

### 5.3 イベントカード / リスト行（/events）★案内係の中核
**リスト表示（主）**: 左に日付ブロック、右に内容の横長行。

```
┌────┬──────────────────────────┐
│ 8月 │ 藤糸づくり体験ワークショップ    [受付中] │
│ 23  │ 📍 遠山郷 和田会館 ・ 💴 2,000円      │
│ (土) │ 遠山ふじ糸 ▸                        │
└────┴──────────────────────────┘
```
- 日付ブロック: 淡藤背景 `#F0EBF7` + 藤紫の日付数字（視線のアンカー）
- 終了イベント: 写真グレースケール + 「終了」バッジ `--color-ended`（アーカイブとして表示は残す）

**月カレンダー（従・切替式）**: セルにイベント日 = 藤紫の結び目ドット。タップで当日のイベントをリスト表示。月移動は左右矢印 + 「今日」ボタン。

### 5.4 体験カード（/experiences）
イベントカードと似た構造だが、日付ブロックの代わりに「随時受付」「要予約」等の受付形態バッジ（success 色）を表示。

### 5.5 工程タイムライン（工芸詳細）★シグネチャー
- 縦 1px の藤紫の糸が上から下へ。各工程に結び目（8px 丸）+ 写真 + 説明
- スクロールで各工程が fade-in-up（Sayo's Journal の stagger パターン流用）

### 5.6 言語スイッチャー
- ヘッダー右端に「日本語 / EN」ピル型トグル。現在 locale を藤紫で表示
- EN 未訳ページに EN で来た場合: 日本語版へフォールバック + 「This page is available in Japanese only」バナー（淡藍背景）

### 5.7 地図
- Google Maps 埋め込み or 静的地図 + 「Google マップで開く」ボタン（お医者さんナビの OS 起動パターン踏襲）
- 体験・イベント詳細では住所テキスト + コピー可能

## 6. ページ別デザイン指針

### ホーム
1. **Hero**: 全幅写真（工芸の手元アップ推奨）+ display 見出し「南信州の手しごとに、会いに行く。/ Meet the crafts of Minami-Shinshu.」+ 検索ではなく**「体験をさがす」「工芸を知る」の 2 大ボタン**（Airbnb の検索ピルは MVP ではオーバースペック）
2. 直近の体験・イベント（リスト行 3 件 + 「すべて見る」）
3. 工芸カード（2 件 + 今後増える前提のグリッド）
4. サイトの目的（「点を線に」を糸モチーフの図で 3 行説明）
5. 最新記事

### 工芸詳細（正本ページ）
- Hero は全幅写真 + 工芸名（写真上に白文字 + 下部グラデーションスクリム）
- 目次（PC: 左サイドに追従 / SP: 上部横スクロールチップ）
- 「体験する」セクションはページ内で最も視覚的に強く（淡藤の面で区切る）

### イベントカレンダー
- デフォルトはリスト表示（開催日昇順）。ヘッダーで「リスト ⇄ カレンダー」切替
- **空の状態**: 「今月の開催予定はありません」で終わらせず、随時受付の体験カードを必ず併記（空白ページ禁止）

### 記事詳細
- Sayo's Journal の記事レイアウトに準拠しつつ、花装飾は入れず糸モチーフの区切りに置換

## 7. モーション

- 基本 transition: 200–300ms ease-out
- スクロール出現: fade-in-up 0.4s（Sayo's Journal の CSS-only パターンを流用、framer-motion 不使用）
- 演出はカード hover と工程タイムラインの 2 箇所に集中させ、他は静的に保つ
- `prefers-reduced-motion: reduce` で全アニメーション無効化

---

## 8. レスポンシブ / アクセシビリティ

- ブレークポイント: SP < 640 / Tablet 640–1024 / PC > 1024
- タップターゲット 44px 以上（高齢ユーザー・観光中の片手操作を考慮）
- フォーカスリング: 藤紫 2px（`:focus-visible`）
- 画像は必ず alt（翻訳テーブルの locale 対応 alt）
- 日付は locale フォーマット（ja: 8月23日(土) / en: Sat, Aug 23）

---

## 9. 写真ガイドライン（紗代さん向け運用メモ）

- イラストではなく実写真のみ。手元・素材・工程のクローズアップを優先
- カード用は 4:3、Hero 用は 16:9 で撮影 or トリミング
- 人物が写る場合は掲載許可を撮影時に確認（交渉時のチェック項目に含める）
- 明るめ・自然光。生成り背景と喧嘩する強い彩度の加工は避ける
