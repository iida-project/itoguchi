# 08 — イベントカレンダー・イベント詳細 ★案内係の中核

**依存**: 02, 04
**参照**: REQUIREMENTS.md §4（カレンダー設計方針）§6 / DESIGN.md §5.3, §6

## 概要

`/[locale]/events`（月表示 ⇄ リスト表示）と `/[locale]/events/[slug]`。本サイトの中核機能。

## 仕様

### 一覧
- **リスト表示が主・月カレンダーが従**。ヘッダーで切替（前面に出す方をフラグで制御できる構造に）
- リスト行: 左に日付ブロック（淡藤背景 + 藤紫の日付数字）、右に タイトル + 受付中バッジ / 会場 / 料金 / 工芸名リンク
- 月カレンダー: 自作グリッド（ライブラリ不使用）。イベント日に藤紫の結び目ドット、タップで当日イベントをリスト表示。月移動は左右矢印 + 「今日」ボタン
- 工芸での絞り込み
- **空の状態**: 「今月の開催予定はありません」で終わらせず、随時受付の体験カードを必ず併記
- 終了イベント: 写真グレースケール + 「終了」バッジでアーカイブ表示（削除しない）
- カレンダーは日次更新で十分（ISR、force-dynamic 不要）

### 詳細（案内係として必須の項目）
日時 / 会場（地図）/ 料金 / 定員・対象 / 持ち物 / **申込方法**（外部リンク or 電話・メール。ボタンラベルは「申し込む（外部サイト）」等、外部遷移を明示）/ 主催（担い手へ）/ 関連工芸
