## 概要

掲載交渉前の 2 団体（遠山ふじ糸伝承の会 / 阿島傘）の仮コンテンツを公開情報から作成し、完成イメージを交渉材料にする。**ルール違反は実害になるため §10 を厳守。**

## 厳守ルール

1. 文章は公開情報ベース。推測箇所（料金・頻度・定員等）は `is_provisional` + 「※確認中」表示
2. **他者の写真は一切使わない**。プレースホルダー枠（生成り背景 + 「ここに◯◯の写真が入ります」）で構成
3. 交渉期間中は本公開しない: 全ページ noindex, nofollow + robots.txt 拒否 + プレビュー URL 共有
4. 交渉成立後: 実素材で置換 → is_provisional 解除 → craft 単位で公開切替。許可記録を管理メモに残す
